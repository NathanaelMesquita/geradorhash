# Gerador de Hash

Este programa gera hashes em uma tabela a partir de arquivos PDF e Excel, utilizando o algoritmo SHA-256. Ele aceita múltiplos arquivos ao mesmo tempo, permitindo uma análise rápida e eficiente.

## Funcionalidades

- Geração de hashes para arquivos PDF e Excel.
- Suporte para múltiplos arquivos de uma só vez.
- Utiliza o algoritmo SHA-256 para garantir a segurança dos hashes.

## Uso

1. Execute o programa.
2. Clique no botão "Carregar Arquivos" para selecionar os arquivos desejados.
3. Após carregar os arquivos, clique em "Gerar PDF com Hashes" para criar um documento PDF contendo a tabela de hashes.

**Observação:** Na primeira vez que executar, o programa pode solicitar permissões de administrador para rodar no seu PC.

## Requisitos

- Python 3.12
